In the previous post, we explored the concept of periodic functions and their period modules. We saw that the period modules of continuous and strictly periodic functions are quite limited, and that for cntinuous functions their sums are periodic only under the very special circumstance of being commensurable.

Outside the world of continuity, we also saw that there are non-constant functions with dense and/or higher-dimensional period modules, and that there are incommensurable strictly periodic functions whose sum is periodic. However, all of the examples we saw were constant almost everywhere.

In this post, we will first find pathological examples of periodic functions that are constant almost nowhere by using the axiom of choice. Next we will see how the wall of measurability prevents us from venturing into the world of pathological periodic functions without abandoning the very useful and intuitive notion of measurability. In future posts, we will then break through this wall anyway, and explore just how crazy periodic functions can get when we assume the axiom of choice and allow them to be non-measurable.

4. Pathological examples using the axiom of choice

Let’s dive right in:

Proposition 4.1: There exists a periodic function \(f\) with a dense period module \(P_f\) such that \(f\) is constant only on sets of measure zero.

Proof: Let \(G\) be a \(\mathbb{Z}\)-submodule of \(\mathbb{R}\) with dimension at least 2 and measure zero (i.e., something like \(\mathbb{Z}_{p,q}\) for incommensurable \(p,q\)), and let \(X\) be a set of representatives of the quotient \(\mathbb{R}/G\) (which exists by the axiom of choice). Define \(f:\mathbb{R}\to\mathbb{R}\) by \(f(x) = r\) if \(x\in r+G\) for some \(r\in X\). This function is periodic with period module containing \(G\), and thus has a dense period module. Moreover, it is constant only on sets of the form \(r+G\), which all have measure zero.

Remark. We can also now find pairs of functions \(f, g\) with incommensurable periods such that \(f+g\) is periodic, and all three functions are constant only on sets of measure zero. To do this, we can simply take \(f\) and \(g\) to be the functions we found in the proof of Proposition 4.1 using \(p\mathbb{Z}\) and \(q\mathbb{Z}\) respectively, where \(p\) and \(q\) are incommensurable. Finally, if we want strictly periodic examples of this phenomenon, we can go back to the \(f, g\) in Example 2.6 and notice that it still works if we replace 0 in the ranges of \(f, g\) with the values of any other function that is periodic in both \(p, q\). Namely, we can replace 0 with the pathological function as in Proposition 4.1 but with \(G = \mathbb{p,q}\). This makes it so that \(f\) and \(g\) are strictly periodic with incommensurable periods, their sum is strictly periodic, and all three functions are constant only on sets of measure zero.

5. The wall of measurability

This brings us to the classic result in measure theory mentioned earlier: if we require our functions to be measurable, then the only functions with dense/higher-dimensional period modules are constant almost everywhere.

This is often proved using the Lebesgue Density Theorem, which states that for any measurable set \(A\subseteq\mathbb{R}\), almost every point of \(A\) is a density point, meaning that the density of \(A\) around that point is 1. This theorem is in turn often proved using the Lebesgue Differentiation Theorem. Using Lebesgue Density, one can show that if a measurable function has a dense set of periods, then it must be constant almost everywhere.

However, given it has been 7 years since I last used any measure theory, I could hardly remember these thoerems when I first approached the problem. And so, I came up with a proof “from scratch,” if you will, and thought it was a fruitful excercise to do so. Here, I will present this proof, which is more elementary in the sense that it does not rely on the Lebesgue Density Theorem or the Lebesgue Differentiation Theorem, but it does rely on some basic properties of measurable sets and functions. In passing, we will also see how this result implies the ergodicity of irrational rotations of the circle, which is a classic result in ergodic theory that naturally connects to the question of periodic functions with dense period modules.

Before we can get to the proof, we need to recall some definitions and facts from measure theory.

5.1 Preliminaries on measure theory

Before diving into the proof, let us recall three fundamental properties of the measure space on the real line that will form the entirety of our logical machinery:

  1. The Measure Space: We work with the standard Lebesgue measure \(m\) on \(\mathbb{R}\), which assigns to any interval \([a,b]\) its familiar length \(m([a,b]) = b-a\). Throughout this proof, all sets and functions discussed are assumed to be Lebesgue measurable.
  2. Translation Invariance: The Lebesgue measure is translation-invariant. If \(E\) is a measurable set and \(t \in \mathbb{R}\), then the translated set \(E+t := \{x+t : x \in E\}\) is also measurable, and \(m(E+t) = m(E)\). This is the geometric engine of our argument, as we will be repeatedly translating level sets by our dense periods.
  3. Approximation by Intervals: While we deliberately avoid local density theorems, we do require a global regularity property. Any measurable set \(E\) of finite measure can be approximated arbitrarily well by a finite union of disjoint open intervals. Formally, for any \(\epsilon > 0\), there exists a finite collection of disjoint open intervals \(I_1, \dots, I_N\) such that the measure of the symmetric difference is bounded: \[m\left(E \triangle \bigcup_{k=1}^N I_k\right) < \epsilon\]

Equipped exclusively with these three basic facts, we can proceed to reconstruct the theorem from the ground up.

5.2 Measurable functions with dense period modules are constant almost everywhere

We are now ready to prove the main result of this section:

Theorem 5.1: If \(f:\mathbb{R}\to\mathbb{R}\) is a measurable function with a dense set of periods, then \(f\) is constant almost everywhere.

[proof will go here] The key idea is to use the density of the set of periods to show that the pre-image of any set under the function must have either full or zero measure. This is a very strong condition, and it will allow us to conclude that the function must be constant almost everywhere. It will also help us establish the ergodicity of irrational rotations of the circle, since we will see that any invariant set under such a rotation must have either full or zero measure: given any invariant set \(A\) for an irrational rotation \(T_\alpha\), the charachteristic function \(\mathbb{1}_{A}\) is periodic with period \(\alpha\) and period \(1\), and thus has a dense set of periods, so \(A\) (which equals the the pre-image of \(\{1\}\) under \(\mathbb{1}_{A}\)) must have either full or zero measure, showing ergodicity of \(T_\alpha\).

Definitions 1.3 (Lebesgue Measure, Measurable Set, Measurable Function) The Lebesgue measure on \(\mathbb{R}\) is a function \(\mu\) that assigns a non-negative real number to certain subsets of \(\mathbb{R}\), called measurable sets, in a way that generalizes the notion of length. A set \(A\subseteq\mathbb{R}\) is measurable if for every \(\epsilon>0\), there exists an open set \(O\) such that \(A\subseteq O\) and \(\mu(O\setminus A)<\epsilon\). A function \(f:\mathbb{R}\to\mathbb{R}\) is measurable if the pre-image of every open set is a measurable set.

As promised, we will dive in to the question of whether a measurable function with a dense set of periods must be constant almost everywhere. We will give an elementary proof of this fact, and in the process, we will also demonstrate another classic result in ergodic theory: that rotations of the circle are ergodic if and only if the angle of rotation is irrational.
